Description §

Design a way to obtain closeness score from KAD-DHT experiments. If needed, the Kad-dht script in libp2p-test-node repository can be modified, but shouldn’t be necessary. PeerID is already exposed in the script already, so we need:

  1. Deploy the experiment.
  2. After the experiment is finished, scrape the logs of all PODs, searching for every PeerID in the network.
  3. In the scripts for the KAD-DHT analysis, add this score as part of the analysis ifself.
  4. Provide a way of visually represent this score. For more information, refer to this document.
